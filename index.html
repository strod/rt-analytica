Perguntas Frequentes

Sob medida. Construímos software feito para o seu negócio. Você é dono do código. Não há mensalidade de plataforma — apenas o projeto de desenvolvimento e suporte contínuo opcional.
A maioria dos projetos leva de 4 a 12 semanas do início até a primeira versão funcional. Um chatbot WhatsApp pode levar 4–6 semanas; um conjunto completo de pipeline de dados e dashboards, 8–12. O escopo depende da complexidade — daremos uma estimativa honesta após a fase de descoberta.
Seus dados continuam sendo seus. Seguimos as melhores práticas de segurança — conexões criptografadas, acesso com menor privilégio, nenhum dado armazenado além do necessário para o sistema funcionar. Assinamos NDAs e trabalhamos dentro dos seus requisitos de conformidade.
Esse é o objetivo. Integramos com ERPs (SAP, Bling, etc.), CRMs, plataformas de anúncios, bancos de dados, planilhas, APIs de mensagens — qualquer coisa que seu negócio utilize. Se tem uma API ou exporta dados, podemos trabalhar com isso.
Oferecemos pacotes de suporte contínuo e iteração. A maioria dos clientes começa com um projeto de construção e migra para um contrato mensal mais leve de manutenção, monitoramento e as inevitáveis melhorias do tipo "dá pra fazer isso também?".
